# ⚡ Power Outage Analysis using Electric Disturbance Dataset

## 📌 Overview
This project analyzes power outage events using the Electric Disturbance Events dataset. The goal is to identify patterns, causes, and impacts of outages using data science techniques.

---

## 📊 Objectives
- Analyze outage trends over time
- Identify major causes of power outages
- Study regional and seasonal patterns
- Evaluate impact based on customers affected and demand loss

---

## 🧠 Methodology
- Data Cleaning and Preprocessing
- Feature Engineering (Year, Month extraction)
- Exploratory Data Analysis (EDA)
- Visualization using Matplotlib & Seaborn

---

## 📈 Key Insights
- Severe weather is the leading cause of outages
- Certain regions show higher outage frequency
- Outages peak during extreme seasonal conditions
- Demand loss correlates with number of customers affected

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
Electric Disturbance Events Dataset (Kaggle)
